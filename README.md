# mySuperheroList
Search your favorite Superhero and find all the juicy details about them .


## Built in

1. React.js
2. MongoDB
3. Express
4. Node
5. Passport for authentication

## Instructions

1. download or clone repo
2. cd into folder
3. yarn install
4. yarn start
5. navigate to localhost 3000 if not automatically redirected
